A dynamic Wordpress sidebar widget to display recently played tracks from the Last.fm service.

== Description ==

Last.fm Sidebar is a Wordpress plugin which allows you to display your last played Last.fm track in your Wordpress sidebar along with album art. It’s fully customisable via the widget manager. Last.fm data is cached in a database as not to hinder the performance of your blog. You can see it in action on [my blog](http://www.peppery.net.nz/ "my blog! :)") ♪

== Installation ==

1. Upload `last.fm.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Appearance -> Widgets and drag the widget into your sidebar
4. Customise the widget's settings, save, and enjoy! :)
